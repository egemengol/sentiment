## How To Run
`numpy` is needed, any reasonably recent version will do. I used 1.18.2

The code needs a data folder like:
```
├── bern.py
├── clas.py
├── data
│   ├── test
│   │   ├── neg
│   │   └── pos
│   └── train
│       ├── neg
│       └── pos
├── measure.py
├── README.md
└── utils.py
```

Run:
```sh
python3 measure.py
```
